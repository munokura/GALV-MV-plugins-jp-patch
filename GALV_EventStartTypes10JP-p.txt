/*:ja
 * @plugindesc 1つのイベントがトリガー3つに異なるイベントを実行できるようにします。
 *
 * @author Galv - galvs-scripts.com
 *
 * @help
 * ---------------------------------------------------------------------------
 *   Galv's Event Start Types
 * ---------------------------------------------------------------------------
 * このプラグインを使用すると、1つのイベントがトリガー3つに
 * 異なるイベントを実行できるようにします。
 * イベントの条件分岐を使用して、
 * イベント毎に異なるコマンドを実行できます。
 * ---------------------------------------------------------------------------
 *  CONDITION BRANCH SCRIPT
 * ---------------------------------------------------------------------------
 *
 *    this.startType(x);    // x は、以下の開始タイプ番号です。
 *                          // 0 - 決定ボタン
 *                          // 1 - プレイヤーから接触
 *                          // 2 - イベントから接触
 *
 * 使用例：
 * ◆条件分岐：スクリプト：this.startType(0); // 決定ボタン
 *
 * ：決定ボタンでのイベント内容
 *
 * ：分岐終了
 * ◆条件分岐：スクリプト：this.startType(1); // プレイヤーから接触
 *
 * ：プレイヤーから接触でのイベント内容
 *
 * ：分岐終了
 *
 * ---------------------------------------------------------------------------
 */
